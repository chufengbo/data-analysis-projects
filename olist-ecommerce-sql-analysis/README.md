# 巴西电商订单经营分析

基于 Olist 巴西电商多表数据，使用 SQLite、SQL 和 Excel 完成销售表现、客户价值、物流体验及卖家集中度分析。

## 项目文件

- `巴西电商订单经营分析报告_正式版.docx`：正式业务分析报告
- `巴西电商订单经营分析.xlsx`：SQL 查询结果与可视化图表
- `巴西电商订单经营分析_SQL代码注释与问题复盘.md`：SQL 代码、注释与问题复盘

## 分析内容

- 月度 GMV 与订单量趋势
- 各州 GMV Top10
- 商品类目 GMV Top10
- RFM 客户分层
- 配送时长与用户评分
- 卖家 GMV 集中度与二八分析

## 技术栈

SQLite、SQL、多表 JOIN、CTE、窗口函数、RFM、Excel
